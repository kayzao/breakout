Experience with Using BRC:


Fantastic!

Not only is it extremely efficient, but the various types of UI elements you can incorporate into your code is tremendously useful. You have the ability to create buttons, sliders, monitors and etc. We believe that this is a great way to have an external source of utilities at your disposal. One slight improvment that can be made is the way you add new UI elements, in order to add a new element, you need to either replace the file with the new additions or you need to copy and paste it.
